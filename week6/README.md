# 极客时间《JAVA进阶训练营》第六周作业

1. **必做作业**：基于电商交易场景（用户、商品、订单），设计一套简单的表结构，提交 DDL 的 SQL 文件到 Github。

   SQL 文件位于 `src\sql` 的 `linjn.sql` 文件。

   总共有数据表 12 个，分别为：

   - **user**：用户表
   - **address**：地址表
   - **category**：商品分类表
   - **tag**：商品标签表
   - **spec**：商品规格表
   - **spec_detail**：商品规格详情表
   - **spu**：spu表
   - **spu_img**：spu 图片表
   - **spu_detail_img**：spu 详细信息图片表
   - **sku**：sku 表
   - **order**：订单表
   - **order_detail**：订单详情表
